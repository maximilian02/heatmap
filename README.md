# Heatmap

## Project Set Up ##

1. 'bower install'
2. 'npm install'
3. 'grunt serve'

## Versiones recomendadas para correr el proyecto ##

- Ruby 1.9.3 
- Compass 0.12.6 (Alnilam)
  - gem install compass -v 0.12.6
- Sass 3.2.19 (Media Mark)
  - gem install sass -v 3.2.19  
- FontAwesome 
  - gem install font-awesome-sass
